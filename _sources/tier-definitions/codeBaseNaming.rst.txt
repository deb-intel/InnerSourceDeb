Code Base Naming
################

1Source migration into approved namespace needed for reuse and contribution to start occurring.

Tier Definitions
****************

.. _tier_codeBaseNaming_start:

* **Tier 5.** Arbitrary Source Control Management (SCM). Project not yet migrated to 1Source Source Control Management (SCM).
* **Tier 4.** Migration to 1Source in progress or may not meet 1Source guidelines. Self defined and maintained code base naming convention.
* **Tier 3.** GitHub project, repository names, and location meet 1Source guidelines. GitHub repos are managed via 1Source inventory. https://github.com/intel-innersource/inventory/blob/master/README.md and https://github.com/intel-innersource/inventory#governance.

.. _tier_codeBaseNaming_end:
