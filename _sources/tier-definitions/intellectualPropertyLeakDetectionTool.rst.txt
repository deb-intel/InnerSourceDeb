Intellectual Property Leak Detection Tool
#########################################

Reuse team needs to be able to leverage previously clean tool results.

Tier Definitions
****************

.. _tier_intellectualPropertyLeakDetectionTool_start:

* **Tier 5.** No IP leak detection.
* **Tier 4.** Manual IP Leak Detection Tool scan passes. IPDLT found in Product Validation tools: http://pvtools.intel.com/
* **Tier 3.** Manual IP Leak Detection Tool scan passes and is executed prior to each public release.
* **Tier 2.** Automated IP Leak Detection Tool scan on defined schedule.
* **Tier 1.** Automated IP Leak Detection Tool scan built into Continuous Integration.

.. _tier_intellectualPropertyLeakDetectionTool_end:
